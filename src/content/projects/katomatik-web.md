The site you're reading. Built with Astro and Tailwind, output as static HTML, packaged
into a container image and served from a home Kubernetes cluster behind a Cloudflare
Tunnel.

The design carries over from the previous katomatik.com — charcoal and whitesmoke, a
retro picture frame around images, and a hand-drawn wavy divider.
